A bot for Ukrainians learning English.  
Generates words and tests based on the user's level (A1–C1).

## About
Smart Word Step is an educational bot designed for Ukrainians who want to improve their English step by step.

## Features
- Generates English words by level
- Tests to check knowledge
- Simple and fast interaction
- Level-based learning (A1–C1)

## How it works
1. The user specifies their level
2. The bot generates words via the Gemini 2.5-flash API
3. The user takes a test
4. Progress improves

## Technology stack
- Python 3.13+
- Telegram Bot API
- AI / NLP
